# 🏦 Bank Marketing Funnel Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Marketing Funnel Analysis Dashboard** built using **Streamlit** and **Plotly**. It analyzes a bank's direct marketing campaign to understand customer conversion behavior, identify drop-off points in the marketing funnel, and provide actionable business insights to improve subscription rates.

The dashboard enables users to explore customer demographics, campaign performance, and conversion metrics through interactive visualizations and filters.

---

## 🎯 Project Objectives

* Analyze customer subscription patterns.
* Measure marketing campaign performance.
* Identify customer drop-off in the marketing funnel.
* Compare subscription rates across different customer segments.
* Generate business insights and recommendations to improve future campaigns.

---

## 📊 Dataset

**Dataset:** Bank Marketing Dataset (UCI Machine Learning Repository)

The dataset contains information collected from direct marketing campaigns conducted by a Portuguese banking institution.

## 🚀 How to Run the Project

1. Clone the repository.

2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit application:

```bash
streamlit run app.py
```

4. Open the local URL generated by Streamlit in your browser.
